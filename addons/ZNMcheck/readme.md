# ZNMcheck

Prints a list of all pop items, trophy items, and key items for ZNM that your character has, including where they can be found in your inventory and for pop items which mob they pop.  
<img width="275" height="137" alt="ZNMcheck" src="https://github.com/user-attachments/assets/2c4a88c5-6465-4661-a99f-973918bb6ba8" />  
Simply type `//znmcheck` or `znmc` to run. No other commands.
