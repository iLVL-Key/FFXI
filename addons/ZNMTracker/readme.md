# ZNMTracker

Tracks all ZNM pop items, trophy items, and key items that your character has in easy to read window. Highlights which NMs are currently available to purchase depending on which Key Items you have. Updates as you go.

## Commands
All commands must be preceded with either `//znmtracker` or `//znmtrack` or `//znmt` (ex: `//znmt tin`)  
- (no command)/show/hide - Show/hide tracker window.
- check/list - List all pop, trophy, and key items this character has and their location.
- tinnin/tin - Display only the path for Tinnin in the tracker window.
- sarameya/sar - Display only the path for Sarameya in the tracker window.
- tyger/tyg - Display only the path for Tyger in the tracker window.
- pan/pw/all - Display all paths in the tracker window.
